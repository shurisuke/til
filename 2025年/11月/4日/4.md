

# Simple Form での collection_radio_buttons の使い方
- f.collection_radio_buttons は、enum の値などを元にラジオボタンを生成できる。
- シンボル（例: :eye_catch_send）を第1引数に、値の配列を第2引数に渡す。
- 第3引数は各値の value 属性にするためのメソッドやラムダ（例: :first, :to_s）。
- 第4引数はラベルとして表示される文字列（例: :last やラムダで日本語化した値など）。

# content_tag の基本と使い所
- content_tag は HTML タグを生成する Rails ヘルパー。
- 例えば iframe を生成したい時に便利。


# Twitter の投稿埋め込み（API不要）
- 埋め込みコードは blockquote + a タグ + widgets.js だけで動作する。
- APIやJavaScript操作は不要、iframeも自動で生成される。


# Slimでのコメントアウト方法
- / を使うとHTMLコメントにはならないが、テンプレート上で無視される。
- /- を使うと Ruby コードの中でも無視される。
