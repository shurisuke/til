# partitionメソッドとは？

partition は、配列を2つのグループに分けるメソッド。
ブロックの評価が true になる要素と false になる要素を、それぞれ別の配列として返す。

### 返り値は2要素の配列（入れ子になった配列）

### 基本構文
true_group, false_group = array.partition { |item| 条件 }

まとめ 返ってくるのは2つの配列を要素に持つ1つの配列 → [true側, false側]
      代入する場合は多重代入で行うと便利
      partition は全要素に対してブロックを評価する（途中で止まらない）


# case式の使い方と使い分け
使い分けの種類

1. 基本構文：値とマッチさせるタイプ (caseに値や変数を渡す)
- case に書いた値（ここでは animal）を === で when の値と比較。
- シンプルな分岐に適している。
case value
when 条件1
  処理1
when 条件2
  処理2
else
  デフォルト処理
end

2. caseに値を渡さないタイプ（条件を直接評価）
- シンプルな分岐に適している。
- 複数の条件を評価するのに向いている。
- case に何も書かないことで、各 when の条件式がそのまま true/false で判定される。
case
when 条件式1
  処理1
when 条件式2
  処理2
end

### 式としても使える！！！
case 式は戻り値を返すことができるので、代入もできる。
grade = case score
        when 90..100 then "A"
        when 80..89  then "B"
        else "C"
        end


# delete メソッドの文字削除
- String#delete("xyz") は、渡された文字をすべて削除する。正規表現ではなく、一文字単位での除去。
- 複数文字を「単語として」除去したい場合は gsub を使う。
