# partitionメソッドとは？

partition は、配列を2つのグループに分けるメソッド。
ブロックの評価が true になる要素と false になる要素を、それぞれ別の配列として返す。

###返り値は2要素の配列（入れ子になった配列）

###基本構文
true_group, false_group = array.partition { |item| 条件 }

まとめ 返ってくるのは2つの配列を要素に持つ1つの配列 → [true側, false側]
      代入する場合は多重代入で行うと便利
      partition は全要素に対してブロックを評価する（途中で止まらない）
