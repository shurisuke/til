# DrivePeek

**DrivePeek（ドライブピーク）は、行き先・食事・時間をまとめて計画できるだけでなく、他の人のドライブプランも参考にできる、“新しいドライブ体験”アプリです。**


## サービス概要
気軽で自由な発想を広げられるドライブプランアプリ。
立ち寄りスポットの登録やメモ機能により、効率的かつ柔軟なプランニングをサポートします。
さらに、AIによるスポット提案機能や、他ユーザーのドライブプランを参考にできる機能により
自分では思いつかなかった選択肢にも出会え、視野を広げた充実のドライブ計画を実現できます。


## このサービスへの思い・作りたい理由
私自身、休日にドライブをするのが好きで、日々のリフレッシュとして大切な時間になっています。
そんな中で、スケジュールを立てている際や実際に出かける時に、いつも次のような悩みや迷いを感じていました。

「途中でどこか知らない場所に立ち寄ってみたいけど、探すのが手間だな…」
「道中で立ち寄る場所で〇〇分滞在して 目的地で〇時間過ごしたら、帰りの時間はどのくらいになるんだろう…」
「行き先で何を食べようか、毎回考えるのに時間がかかる…」

こうした日々の体験や思いから、
情報収集・プラン作成・時間管理を丸ごとサポートしてくれて、ドライブ体験を充実させるアプリがあったら良いのに」
という気持ちが高まり、今回このドライブプランアプリ「Drivepeek（仮）」の開発を決意しました。


## ユーザー層について
1. 自然やごはんが好きでドライブを楽しむ人
自然の景色を眺めたり、おいしいごはんを目的に出かけたりと、ドライブそのものを楽しむ層は、行き先選びや寄り道の体験が旅の満足度を大きく左右します。
そのため、スポット情報の整理やルート検討、時間配分などを一括でサポートできる本サービスは、この層のニーズと特に相性が良いと考え、この層を対象にしました。

2. 他人の体験からアイデアを得たい人
行き先やルートの発想が浮かばないとき、他人の具体的な体験が参考になることは多々あると思います。
そのため、他ユーザーのプランを閲覧・保存できる機能を通じて、アイデアを得たい層に新たな選択肢を提供できると考え、この層を対象にしました。

3. 行き先や食事の選択に迷いがちな人
「ドライブはしたいけど、どこへ行こう？何を食べよう？」という悩みは、多くの人が一度は経験する身近な迷いです。
こうした層は、選択肢が多いがゆえに決めきれず、行動までのハードルが上がってしまうことも、考えられます。
そのため、「自分では思いつかなかった選択肢」と出会える体験を通じて、視野を広げながらスムーズにプランを立てられるようサポートしたいと考え、この層を対象にしました。

4. 遠出したいけど時間が限られている人
忙しい中でも無理なく遠出したい人にとって、「滞在時間」や「移動時間」を踏まえた計画づくりは大きな負担になります。
この層は特にスケジュール管理のニーズが高く、時間を考慮した計画立てをサポートする本アプリの価値を強く感じてもらえると考え、この層を対象にしました。


## サービスの利用イメージ
ユーザーは、埋め込み地図を見ながら行きたいスポットを検索・保存し、滞在時間やメモを記入することで、効率的かつ具体的なドライブスケジュールを簡単に作成できます。
他のユーザーのプランを参考にしたり、AIによるおすすめスポット提案を受けることで、「行き先に迷う時間」を減らしながら、視野を広げたプランニングが可能になります。
地図機能とAI提案、さらに他ユーザーのプランから得られるインスピレーションを一体化することで、より充実したドライブ体験を提供します。


## ユーザーの獲得について
- 他人の体験からアイデアを得たい人
他ユーザーのドライブプランを一覧・検索・閲覧できる機能や、いいね・ブックマーク機能により、匿名でも気軽に他人のアイデアを取り入れ、自分らしいプランづくりに活かせます。

- 遠出したいけど時間が限られている人
滞在時間や移動時間を考慮し、帰宅時間までを含めたスケジュールを視覚的に構築できる機能により、「限られた時間内でも安心して遠出できる」体験をサポートします。

- 自然やごはんが好きでドライブを楽しむ人
Google Maps APIとAI提案機能により、自分では気づけなかったスポットや話題の飲食店・観光地と出会える体験を提供。登録・検索・保存など直感的な地図操作で、新たなドライブの楽しみ方を広げます

- 行き先や食事の選択に迷いやすい人
　キーワードや好みからAIがスポットを提案し、結果を地図上に視覚化。ドライブプラン作成のハードルを下げ、迷いの時間を削減することで、スムーズな意思決定をサポートします。


## サービスの差別化ポイント・推しポイント
- 他のユーザーのドライブプランから気軽にインスピレーションを得られる
- 地図・AI・プラン参照機能の三位一体で、これまでにない“ドライブ体験”を実現
- 出発時間・滞在時間・移動時間を元に帰宅まで見通せる、一歩踏み込んだスケジューリング機能


## 機能候補
- MVPリリース
1. スポットの登録・閲覧・検索機能
- Google Maps API を用いて、ユーザーが地図上からスポットを検索・選択・登録できる。
- 登録されたスポット情報（名称・住所・緯度経度）は都道府県別に一覧表示可能。（検索は、キーワード・カテゴリ・都道府県など複数条件で絞り込みが可能。）

2. Google Maps API 連携
- 地図上でスポットを検索し、選択した場所をアプリ内の目的地データとして保存可能

3. ドライブプラン作成・保存機能
- 複数のスポットを組み合わせて「ドライブプラン」を作成・保存できる。（スポットの並べ替え可能）
- 各プランにタイトル・メモ・都道府県（複数選択可能）を追加する機能

4. 滞在時間とスケジュール管理機能
- 各スポットに「滞在時間（分）」を設定。
- スポット間の移動時間はGoogle Maps Directions APIを使って算出。
- 出発時間を設定することで、帰宅予定時間を自動計算。
- スケジュールを視覚化し、わかりやすく把握できるようにする。

5. ユーザー登録機能（Sorcery）
- Sorcery を用いたシンプルなログイン機能。
- ログインしたユーザーのみがスポット・プランの作成・編集・削除が可能。
- ゲスト閲覧用に公開されている他ユーザーのプランを見れる。

- 本リリース
1. 他ユーザーのドライブプランの閲覧・いいね機能
- 他のユーザーが作成したドライブプランを閲覧可能に。
- 気に入ったプランに「いいね」で反応できる。
- 気に入ったプランの「ブックマーク機能」を作成する
- 投稿者は、プランごとに「ユーザー名を表示する / 匿名にする」を選択可能。(匿名モードではユーザーIDや投稿者名を伏せた状態で表示される。)

2. 都道府県別検索機能
- ドライブプラン作成時に設定した都道府県をもとに、地域ごとのドライブプラン一覧を検索・閲覧可能に。

3. AIによるドライブスポット提案（自然文入力）
- ChatGPT API等を使い、ユーザーに入力してもらった文章から命令文を作成して、ドライブスポットを提案する機能。（提案結果を地図に自動反映し、複数表示させる。）

4. SNS共有機能

5. タグ機能（季節・シーンなど）


## 使用する技術スタック
### MVPリリース
- 使用フレームワーク : Ruby on Rails
- データベース : PostgreSQL
- デプロイ : Render
- 検索機能 : gem 'Ransack'
- 認証ライブラリ : gem 'Sorcery'
- 地図API :
  - Google Maps JavaScript API （動的な地図表示）
  - Google Maps Places API（スポット検索）
  - Google MapsDirections API（経路計算・移動時間）

### 本リリース
- AI連携 : OpenAI API
- タグ機能 : gem 'ActsAsTaggableOn'
- SNS共有 : navigator.share API
